Git与TFS最大区别在于：
git 通过在一套代码上维护不同的指向标识，所以创建分支及合并分支等变得容易（改变指针的指向即可）
而传统的集中式源码管理采用的文件的复制粘贴（比如从TFS上拉取不同的分支将有对应的分支文件夹出现在本地，含有大量冗余的信息）
